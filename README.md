# EvA

Aplicação web para o controle de eventos acadêmicos  

**Preparação**

Criar ambiente virtual:  

```
python3 -m venv venv
```

Entrar no ambiente virtual:  

```
source venv/bin/activate
```

**Depedências**

Instalar as dependências:  

```
pipenv install
```

**Execução**

Rodar aplicação:  

```
python3 manage.py runserver
```
